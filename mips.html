MIPS CPU Simulator

Computer Architecture - C++, MIPS Assembly Language

I created a MIPS CPU simulator designed to execute MIPS-1 big-endian binaries for my Computer Architecture module. I implemented the simulator in C++ thereby developing my proficiency with the language. I also became comfortable using and simulating execution of a plethora of MIPS Assembly Language instructions.

In addition to this I also generated an extensive testbench using MIPS instructions and utilized bash to create an automated script for testing the simulator.
