Tile Ninja

FPGA Image Processing - C++, HDL, Python

This project focused on creating an interactive application making use of motion control features by using a Field Programmable Gate Array (FPGA) to carry out real time video processing. I, as part of a team, used the Xilinx Digilent PYNQ-Z1 board and Xilinx Vivado Design Suite to synthesise a Hardware Description Language (HDL) implementation and Jupyter Notebook to create our python-based game interface.

The aim was to utilise an FPGA to enable gesture control of a game and eliminate the need for other peripherals such as a keyboard or mouse during interaction with the game. The game was designed to take in input the coordinates of the hand (specifically the glove that is worn by the user) via colour detection and calculate the average position of the selection of pixels to track the hand’s movement during gameplay. Output frames are written with black tiles moving down the screen and the coordinates of each tile are tracked to determine when a collision occurs with the hand. Accurate detection and stability were of great importance for this idea as a smooth, seamless experience was required without compromising frame rate or hardware performance.

To achieve the final product I developed my working knowledge of C++ and optimisation for hardware, and my understanding of how to use FPGA’s and programmable logic to process video frames.
